# Notebooks

## Brain_Tumor_Detection_CNN.ipynb

This is the main Jupyter notebook containing the complete brain tumor detection pipeline.

### How to use:

1. Download the notebook from this folder
2. Open it in Jupyter Notebook or Google Colab
3. Follow the cells in order to:
   - Download and prepare datasets from Kaggle
   - Preprocess and visualize MRI images
   - Build and train the CNN model
   - Evaluate performance with test set
   - Visualize predictions and metrics

### Requirements:
- See `requirements.txt` in the root directory
- GPU recommended for faster training
